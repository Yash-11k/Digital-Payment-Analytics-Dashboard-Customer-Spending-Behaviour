# Digital Payment Analytics Dashboard

This project analyzes customer spending patterns and digital payment trends using Power BI.  
The dashboard provides insights into revenue distribution, transaction behavior, and customer segmentation.

## Project Objective
To understand how customers use digital payment systems and identify key revenue drivers such as card type, spending category, and payment method.

## Dataset
The dataset contains transactional data including:

- Customer ID
- Gender
- Income Group
- Education Level
- Card Category
- Transaction Amount
- Interest Earned
- Spending Category
- Payment Method
- Transaction Date

## Key KPIs
The dashboard tracks several important business metrics:

- Total Revenue: **55M**
- Total Transaction Amount: **45M**
- Total Customers: **10K**
- Interest Earned: **7.84M**

## Key Insights
Some major insights from the analysis:

- Bills and entertainment are the **top spending categories**
- Platinum and Gold cards contribute the **highest revenue**
- High income group customers generate the **largest share of revenue**
- Swipe transactions dominate over chip and online payments
- Revenue remains relatively stable across quarters

## Dashboard Features
The Power BI dashboard includes:

- Customer segmentation analysis
- Spending category breakdown
- Revenue trends over time
- Card category performance
- Payment method analysis
- Interactive filters for gender, week, and quarter

## Tools Used

- Power BI
- DAX
- Data Visualization
- Data Cleaning
- Business Analytics

## How to Use

1. Download the `.pbix` file
2. Open it using Power BI Desktop
3. Explore the interactive dashboard using filters and slicers

## Project Outcome

This dashboard helps businesses understand customer behavior, optimize payment strategies, and identify revenue opportunities through data-driven insights.
